# Kontadm

Kontena Kubernetes installer, a simple, fast Kubernetes installer that works everywhere.

## Requirements

- Minimal Ubuntu 16.04 nodes with SSH access
- Swap disabled on all nodes

## Usage

```
$ kontadm up -c cluster.yml
```
